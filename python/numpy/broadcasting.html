Broadcasting

NumPy's magical shape matching

Like a Public Address System

Imagine a school principal speaking through a PA (Public Address) system. Even though there is only one principal speaking (a single value), their voice is broadcasted to every single classroom (the array) simultaneously.

In mathematics, adding a matrix to a single number shouldn't technically work without loops. But NumPy performs Broadcasting: it virtually stretches out the smaller object to match the shape of the larger object, so the mathematical operation can run smoothly.

Real-Life Analogy

Applying a uniform standard: Suppose you have an array of 5 students' marks. To add 5 bonus marks to everyone, you don't need to create an array of `[5, 5, 5, 5, 5]`. You just do `marks + 5`. NumPy broadcasts the `5` across all elements.

broadcasting.py 
import numpy as np

marks = np.array([70, 85, 62])

# The single integer '5' is broadcasted to all items
print(marks + 5) 
# Output: [75 90 67]
📏

The Rule of Broadcasting

Broadcasting isn't unlimited. When operating on two arrays, NumPy compares their shapes starting from the rightmost dimension. Two dimensions are compatible if:

  • They are exactly equal, OR
  • One of them is 1
