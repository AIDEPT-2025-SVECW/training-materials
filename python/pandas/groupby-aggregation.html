Like Sorting Papers by Section

Imagine a teacher has exam papers for Class 10 — Sections A, B, and C all mixed together. Before calculating the class average, she first sorts all papers by section, groups them into three piles, and then calculates the average for each pile separately.

That is exactly what groupby() does in Pandas. It splits the DataFrame by a category column, applies a function (like sum, mean, count) to each group, and combines the results back.

Real-Life Analogy — Split-Apply-Combine

Step 1 (Split): Separate the sales records by State (AP, TN, MH).
Step 2 (Apply): Calculate total sales for each state.
Step 3 (Combine): Get one tidy summary table.

groupby.py 
import pandas as pd

df = pd.DataFrame({
    "State":  ["AP", "TN", "AP", "TN", "MH"],
    "Sales":  [200, 150, 300, 100, 400],
    "Units":  [20, 15, 30, 10, 40]
})

# Total sales per state
state_sales = df.groupby("State")["Sales"].sum()
print(state_sales)

# Multiple aggregations at once
summary = df.groupby("State").agg({
    "Sales": ["sum", "mean"],
    "Units": "sum"
})
print(summary)

Common Aggregation Functions

Function What It Computes
sum()Total of all values in the group.
mean()Average value in the group.
count()Number of non-NaN rows in the group.
max() / min()Highest / Lowest value in the group.
std()Standard deviation — spread of values.
