CLI Usage Examples

Create a Simple Web Application

app.groovy 
@RestController
class HelloController {
    @RequestMapping("/")
    String hello() {
        return "Hello, Spring Boot CLI!"
    }
}

@SpringBootApplication
class Application {
    static void main(String[] args) {
        SpringApplication.run(Application, args)
    }
}
Terminal 
# Run the application
spring run app.groovy

# Access the application at http://localhost:8080

Common CLI Commands

spring --version Show CLI version
spring run app.groovy Run a Groovy script
spring run app.groovy -- --server.port=8081 Run with custom port
spring jar app.jar app.groovy Create executable JAR
spring test app.groovy Run tests
spring grab dependencies.groovy Download dependencies
