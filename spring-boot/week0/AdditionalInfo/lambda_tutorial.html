üéØ What are Lambda Expressions?

Lambda expressions are a feature introduced in Java 8 that allows you to write more concise and functional-style code. Think of them as anonymous functions - functions without a name that can be passed around as values.

Simple Definition: A lambda expression is a short way to write a function that can be passed as an argument to another method.

‚ùå Old Way (Anonymous Class)

// Traditional anonymous class
Runnable r = new Runnable() {
    public void run() {
        System.out.println("Hello World");
    }
};

‚úÖ New Way (Lambda)

// Lambda expression
Runnable r = () -> System.out.println("Hello World");

üìù Lambda Syntax

Lambda expressions have a simple syntax:

(parameters) -> { body }

üîß Functional Interfaces

Lambda expressions work with functional interfaces - interfaces that have exactly one abstract method.

Key Point: A functional interface can have multiple methods, but only ONE abstract method.
