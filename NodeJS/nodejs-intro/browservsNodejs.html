üåê Browser JavaScript

Environment

Runs in web browsers (Chrome, Firefox, Safari, etc.)

Global Object

// Available in browser
console.log(window); // Global object
console.log(document); // DOM access

Available APIs

  • window - Global object
  • document - DOM manipulation
  • fetch() - HTTP requests
  • localStorage - Storage (not in Claude artifacts)
  • addEventListener() - Events
  • alert(), prompt() - User dialogs

Try Browser APIs:

Code Example

// Browser-specific code
const button = document.querySelector('button');
button.addEventListener('click', () => {
  window.alert('Hello from browser!');
});

// Fetch from API
fetch('https://api.example.com/data')
  .then(res => res.json())
  .then(data => console.log(data));

üü¢ Node.js

Environment

Runs on servers and local machines (outside browser)

Global Object

// Available in Node.js
console.log(global); // Global object
console.log(process); // Process info
// No window or document!

Available APIs

  • global - Global object
  • process - Process control
  • require() - Module system
  • fs - File system access
  • http/https - Server creation
  • path - Path manipulation

Code Example

// Node.js-specific code
const fs = require('fs');
const http = require('http');

// Read file from disk
fs.readFile('data.txt', 'utf8', (err, data) => {
  console.log(data);
});

// Create HTTP server
const server = http.createServer((req, res) => {
  res.writeHead(200);
  res.end('Hello from Node!');
});
server.listen(3000);
