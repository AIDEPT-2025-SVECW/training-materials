What is Mongoose?

Mongoose is an Object Data Modeling (ODM) library for MongoDB and Node.js. Think of it as a bridge between your Node.js application and MongoDB database.

Why do we need Mongoose?

  • Schema Definition: MongoDB is schema-less, but Mongoose lets you define structure for your data
  • Validation: Built-in validation for data before saving to database
  • Type Casting: Automatically converts data to correct types
  • Query Building: Easier and more readable query syntax
  • Middleware Support: Run code before/after certain operations
  • Custom Methods: Add your own helper methods to models (which we'll demonstrate!)
Instance Methods

Instance Methods

Methods that work on individual documents (instances). Called on a specific student object. 

// Schema Definition
const studentSchema = new Schema({
  name: String,
  marks: [Number]
});

// Instance Method
studentSchema.methods.calculateAverage = function() {
  const sum = this.marks.reduce((a,b) => a+b, 0);
  return sum / this.marks.length;
};

// Usage
const priya = await Student.findOne({name: 'Priya'});
priya.calculateAverage(); // Called on instance

Static Methods

Static Methods

Methods that work on the entire Model/Collection. Called directly on the Student Model. 

// Static Method
studentSchema.statics.findToppers = function(minAvg) {
  return this.find({
    averageMarks: { $gte: minAvg }
  }).sort({ averageMarks: -1 });
};

// Usage
const toppers = await Student.findToppers(85);
// Called on Model itself, not instance
üí° Use static methods for operations that affect multiple documents or the entire collection, like finding, counting, or bulk operations.

Query Helpers

Query Helpers

Chainable query modifiers. Extend Mongoose's query building capabilities. 

// Query Helper
studentSchema.query.byCity = function(city) {
  return this.where({ city: city });
};

studentSchema.query.passed = function() {
  return this.where({ averageMarks: { $gte: 40 } });
};

// Usage - Chain multiple helpers!
const students = await Student
  .find()
  .byCity('Mumbai')
  .passed()
  .sort({ name: 1 });
üí° Use query helpers to create reusable, chainable query filters. They make your code more readable and maintainable.
