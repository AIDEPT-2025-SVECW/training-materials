üìñ What is the OS Module?

The Node.js OS module is like having a friendly interpreter who can tell you everything about the computer your Node.js application is running on. Just like how you might ask someone "What kind of phone do you have?" or "How much battery do you have left?", the OS module lets your code ask the computer similar questions.

Real-World Analogy

Imagine the OS module as a computer health inspector. Just like a health inspector can tell you about a restaurant's kitchen conditions, staff, and equipment, the OS module can tell you about your computer's operating system, memory, CPU, and other vital statistics. It's your application's way of understanding its environment!

// Basic import of OS module
const os = require('os');

// The OS module is built into Node.js
// No need to install it separately!

üõ†Ô∏è Key OS Module Methods

os.platform() - Operating System Type

This method tells you what type of operating system is running, like asking "Are you Windows, Mac, or Linux?"

const os = require('os');

console.log('Platform:', os.platform());
// Possible outputs: 'win32', 'darwin', 'linux', 'freebsd', etc.

os.arch() - CPU Architecture

Tells you the CPU architecture, like asking "Are you built for 32-bit or 64-bit processors?"

console.log('Architecture:', os.arch());
// Common outputs: 'x64', 'arm', 'arm64', 'ia32'

os.release() - OS Version

Gets the operating system release version, like asking "What version are you running?"

console.log('OS Release:', os.release());
// Example output: '10.0.19041' (Windows 10)

os.totalmem() - Total Memory

Returns total system memory in bytes. Think of it as asking "How much RAM does this computer have in total?"

const totalMemory = os.totalmem();
console.log('Total Memory:', Math.round(totalMemory / 1024 / 1024 / 1024), 'GB');
// Converts bytes to gigabytes for easier reading

os.freemem() - Available Memory

Shows how much memory is currently available, like checking "How much RAM is free right now?"

const freeMemory = os.freemem();
const totalMemory = os.totalmem();
const usedPercent = ((totalMemory - freeMemory) / totalMemory * 100).toFixed(1);

console.log('Free Memory:', Math.round(freeMemory / 1024 / 1024), 'MB');
console.log('Memory Usage:', usedPercent + '%');

os.cpus() - CPU Information

Returns detailed information about each CPU core, like getting a full report card for your processor.

const cpus = os.cpus();
console.log('Number of CPU cores:', cpus.length);
console.log('First CPU details:', {
    model: cpus[0].model,
    speed: cpus[0].speed + ' MHz'
});

os.networkInterfaces() - Network Adapters

Lists all network interfaces, like asking "What are all the ways this computer can connect to networks?"

const interfaces = os.networkInterfaces();
Object.keys(interfaces).forEach(name => {
    console.log(`Interface: ${name}`);
    interfaces[name].forEach(info => {
        if (info.family === 'IPv4' && !info.internal) {
            console.log(`  IP Address: ${info.address}`);
        }
    });
});

os.userInfo() - Current User

Gets information about the currently logged-in user, like asking "Who is using this computer right now?"

const userInfo = os.userInfo();
console.log('Current user:', {
    username: userInfo.username,
    homedir: userInfo.homedir,
    shell: userInfo.shell
});

os.hostname() - Computer Name

Returns the hostname of the computer, like asking "What's this computer's name on the network?"

console.log('Hostname:', os.hostname());
// Example output: 'MyComputer' or 'server-01'

os.homedir() - Home Directory

Returns the path to the current user's home directory, like asking "Where is my personal folder?"

console.log('Home directory:', os.homedir());
// Windows: 'C:\\Users\\username'
// macOS/Linux: '/home/username' or '/Users/username'

os.tmpdir() - Temporary Directory

Gets the path to the temporary directory, like asking "Where should I put files that I only need briefly?"

console.log('Temp directory:', os.tmpdir());
// Windows: 'C:\\Users\\username\\AppData\\Local\\Temp'
// macOS/Linux: '/tmp'

os.uptime() - System Uptime

Returns how long the system has been running in seconds, like asking "How long since you were last restarted?"

const uptime = os.uptime();
const days = Math.floor(uptime / 86400);
const hours = Math.floor((uptime % 86400) / 3600);
const minutes = Math.floor((uptime % 3600) / 60);

console.log(`Uptime: ${days}d ${hours}h ${minutes}m`);

üéØ Practical Use Cases

System Monitoring Application

Create a simple system monitor that checks various system metrics:

const os = require('os');

function getSystemStatus() {
    const totalMem = os.totalmem();
    const freeMem = os.freemem();
    const usedMem = totalMem - freeMem;
    
    return {
        platform: os.platform(),
        architecture: os.arch(),
        hostname: os.hostname(),
        uptime: formatUptime(os.uptime()),
        memory: {
            total: formatBytes(totalMem),
            used: formatBytes(usedMem),
            free: formatBytes(freeMem),
            usagePercent: ((usedMem / totalMem) * 100).toFixed(1)
        },
        cpus: os.cpus().length,
        loadAverage: os.loadavg() // Unix-like systems only
    };
}

function formatBytes(bytes) {
    const sizes = ['B', 'KB', 'MB', 'GB'];
    if (bytes === 0) return '0 B';
    const i = Math.floor(Math.log(bytes) / Math.log(1024));
    return Math.round(bytes / Math.pow(1024, i) * 100) / 100 + ' ' + sizes[i];
}

function formatUptime(seconds) {
    const days = Math.floor(seconds / 86400);
    const hours = Math.floor((seconds % 86400) / 3600);
    const minutes = Math.floor((seconds % 3600) / 60);
    return `${days}d ${hours}h ${minutes}m`;
}

// Usage
console.log(getSystemStatus());

Cross-Platform Path Handling

Use OS module to handle paths correctly across different operating systems:

const os = require('os');
const path = require('path');

function getConfigPath() {
    const platform = os.platform();
    const homedir = os.homedir();
    
    switch(platform) {
        case 'win32':
            return path.join(homedir, 'AppData', 'Local', 'MyApp');
        case 'darwin':
            return path.join(homedir, 'Library', 'Application Support', 'MyApp');
        default: // Linux and others
            return path.join(homedir, '.config', 'MyApp');
    }
}

console.log('Config directory:', getConfigPath());
