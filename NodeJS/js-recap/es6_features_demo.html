Template Literals

What are Template Literals?

Template literals are a way to create strings in JavaScript using backticks (`) instead of quotes. They allow you to embed expressions directly into strings using ${} syntax, making string creation more readable and powerful.

Analogy: Think of template literals like fill-in-the-blank forms. Instead of writing separate pieces and gluing them together, you create a template with placeholders that get filled in automatically. It's like having a mad lib where the computer fills in the blanks for you!

‚ùå Old Way (String Concatenation)

let name = "John";
let age = 25;
let message = "Hello, my name is " + name + 
              " and I am " + age + " years old.";
console.log(message);

‚úÖ New Way (Template Literals)

let name = "John";
let age = 25;
let message = `Hello, my name is ${name} 
               and I am ${age} years old.`;
console.log(message);

Spread Operator (...)

What is the Spread Operator?

The spread operator (...) allows you to expand arrays, objects, or other iterables into individual elements. It's like unpacking a suitcase - taking all the items out and spreading them around.

Analogy: Imagine you have a box of marbles (an array). The spread operator is like dumping all the marbles out of the box so you can see and use each marble individually. Or think of it as "unpacking" - when you unpack a suitcase, you spread all the contents out.

‚ùå Old Way

// Combining arrays
let arr1 = [1, 2, 3];
let arr2 = [4, 5, 6];
let combined = arr1.concat(arr2);

// Finding max in array
let numbers = [10, 5, 15, 3];
let max = Math.max.apply(Math, numbers);

‚úÖ New Way (Spread Operator)

// Combining arrays
let arr1 = [1, 2, 3];
let arr2 = [4, 5, 6];
let combined = [...arr1, ...arr2];

// Finding max in array
let numbers = [10, 5, 15, 3];
let max = Math.max(...numbers);

Advanced Examples

Combining Both Features

Template literals and spread operators work great together! You can use template literals to create dynamic strings while using spread operators to work with the data.

function generateReport() {
    const names = getArrayFromInput('studentNames');
    const grades = getNumberArrayFromInput('grades');
    
    // Using spread to find min/max
    const highest = Math.max(...grades);
    const lowest = Math.min(...grades);
    const average = grades.reduce((sum, grade) => sum + grade, 0) / grades.length;
    
    // Using template literals for the report
    const report = `
üìä CLASS REPORT
================
üë• Students: ${names.length}
üìà Highest Grade: ${highest}
üìâ Lowest Grade: ${lowest}
üìä Average: ${average.toFixed(2)}

üèÜ Top Performer: ${names[grades.indexOf(highest)]} (${highest})
üìö Students: ${names.join(', ')}
    `.trim();
    
    return report;
}
