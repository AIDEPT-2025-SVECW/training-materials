Setup & Hello World

Install the tools you need and write your very first TypeScript program.

What are Node.js & NPM?

Before we can run TypeScript, we need to install Node.js.

🟢 Node.js

Traditionally, JavaScript only ran inside your web browser. Node.js is a program that lets you run JavaScript on your computer (like a normal application or a backend server). We need Node.js to run the TypeScript compiler.

📦 NPM (Node Package Manager)

NPM comes installed automatically with Node.js. It is a massive library of pre-written code and tools. We will use NPM to download and install the TypeScript compiler onto your computer.

Step-by-Step Setup

1 Install Node.js

If you don't have Node.js installed, download it from the official website. Choose the LTS (Long Term Support) version.

Download Node.js

To verify it installed correctly, open your terminal (Command Prompt/PowerShell/Mac Terminal) and type node -v. It should print a version number.

2 Install TypeScript

Now that you have NPM, use it to install TypeScript globally so you can use it from anywhere on your computer.

Terminal
npm install -g typescript

Mac/Linux users might need to run sudo npm install -g typescript. Verify the installation by running tsc -v.

Your First TypeScript Program

Let's write, compile, and run our first script.

3 Create the file

Create a new folder somewhere on your computer, open it in VS Code, and create a file named hello.ts.

hello.ts
// We are declaring that 'message' must be a string
let message: string = "Hello, TypeScript World!";

console.log(message);

4 Compile to JavaScript

Browsers and Node.js don't understand .ts files directly. We must ask the TypeScript compiler (tsc) to translate it into standard JavaScript.

Terminal
tsc hello.ts

Look in your folder! A new file named hello.js has magically appeared. This is the compiled JavaScript.

5 Run the Program

Finally, use Node.js to run the compiled JavaScript file.

Terminal
node hello.js

> Hello, TypeScript World!

🎉 Congratulations! You've just written and executed your first TypeScript program!
