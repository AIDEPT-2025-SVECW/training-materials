1. Font-Style Property

Concept: Controls the slant or style of text‚Äînormal, italic, or oblique (artificially slanted).
Think of handwriting on a train: normal is when the train is stationary (upright), italic is when the train tilts naturally on curves (designed slant), and oblique is when you tilt your notebook artificially while the train is moving straight!

2. Text-Decoration Property

Concept: Adds visual lines to text‚Äîunderline, overline, line-through, or none. Can control line style, color, and thickness.
Imagine marking exam papers: underline highlights important points, line-through strikes incorrect answers, overline marks exceptional work at the top, and none leaves clean unmarked text!

3. Text-Transform Property

Concept: Changes the capitalization of text‚Äîuppercase, lowercase, capitalize (first letter of each word), or none.
Like addressing letters: uppercase is shouting on a loudspeaker (JAI HIND!), lowercase is whispering softly, capitalize is formal title case for names (Mr. Ramesh Kumar), and none keeps original typing intact!

.heading {
    text-transform: uppercase;  /* ALL CAPS FOR HEADINGS */
}

.email {
    text-transform: lowercase;  /* emails should be lowercase */
}

.title {
    text-transform: capitalize; /* Each Word Starts With Capital */
}

.preserve {
    text-transform: none;       /* Keep original case */
}

4. Text-Align Property

Concept: Controls horizontal alignment of text within its container‚Äîleft, right, center, or justify (stretch to fill width).
Picture parking cars in marked spots: left align is all cars parked on the left side, right align parks them on the right, center align places each car in the middle, and justify stretches each car to fit perfectly between both lines!

Left Align

Default for LTR languages. Text starts from left edge.

Right Align

Default for RTL languages. Text aligns to right edge.

Center Align

Perfect for headings and short statements.

Justify

Stretches text to create clean edges on both sides, commonly used in newspapers and books. 

/* Text-Align CSS Examples */
p {
    text-align: left;       /* Default alignment */
}

.center-heading {
    text-align: center;     /* Center headings */
}

.invoice-total {
    text-align: right;      /* Right-align numbers */
}

.paragraph {
    text-align: justify;    /* Book-style justified text */
    text-align-last: left;  /* Last line aligns left */
}
