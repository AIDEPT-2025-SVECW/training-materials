1. Grid Container & Grid Items

Concept: Grid turns an element into a 2D layout system where children become grid items automatically positioned in rows and columns.
Analogy: Think of a cricket stadium's seating arrangement. The stadium (container) has sections, rows, and seat numbers. CSS Grid is like the stadium management system that defines how many sections (columns), rows, and the spacing between seats (gaps) exist. Each spectator (grid item) gets assigned to a specific seat automatically or can choose premium seats (custom positioning).

3. Defining Rows & Columns - Unit Types

Concept: Grid supports multiple unit types: fr (fractional units), px (pixels), % (percentages), auto, and minmax() for flexible sizing.
Analogy: Imagine dividing a paratha among family members. fr units are like "fair shares" - 2fr gets double of 1fr. px is like "Dadi gets exactly 100g, no compromise". auto is like "take what you need based on your appetite". minmax() is "at least 50g, but can have up to 200g if available". Grid distributes space accordingly!

üéÆ Interactive Demo: Column Unit Comparison

Fractional Units (fr)

Explanation: The available space is divided into fraction units. If you set 1fr, 2fr, 1fr - the total is 4 units. Col 2 gets 2/4 (50%) of space, while Col 1 and 3 each get 1/4 (25%).

Mixed Units

Pattern: 200px | 1fr | 2fr | 150px

First and last columns are fixed width. Middle columns share remaining space in 1:2 ratio.

MinMax Function

Pattern: minmax(150px, 1fr) | minmax(200px, 2fr) | minmax(100px, 1fr)

Each column has minimum width but can grow. Resize browser to see effect!

Auto Sizing

Pattern: auto | 1fr | auto

First and last columns size to fit content. Middle column takes remaining space.

/* Different Unit Types */

/* 1. Fractional Units - Proportional Space */
grid-template-columns: 1fr 2fr 1fr;  /* 25% 50% 25% of available space */

/* 2. Fixed + Flexible */
grid-template-columns: 200px 1fr 150px;  /* Fixed sidebars, flexible content */

/* 3. MinMax - Responsive Boundaries */
grid-template-columns: minmax(200px, 1fr) minmax(300px, 2fr) minmax(150px, 1fr);
/* Each column: minimum 200/300/150px, but can grow */

/* 4. Auto - Content-based */
grid-template-columns: auto 1fr auto;  /* Sides fit content, center expands */

/* 5. Repeat Function */
grid-template-columns: repeat(4, 1fr);  /* Same as: 1fr 1fr 1fr 1fr */
grid-template-columns: repeat(3, minmax(200px, 1fr));  /* 3 responsive columns */

4. Gaps - Spacing Between Items

Concept: Gaps create consistent spacing between grid items without using margins. Use gap for both, or row-gap and column-gap separately.
Analogy: Like the spacing between tiles in a bathroom floor. The tiles are your grid items, and the grout lines are the gaps. You can make all gaps the same width, or have wider gaps between rows than columns (like having thicker horizontal grout lines). Gaps don't affect the outer edges - no gap between grid and container wall!

5. Responsive Grid Design

Concept: CSS Grid provides powerful responsive tools: auto-fit, auto-fill, minmax(), and media queries to create layouts that adapt seamlessly to any screen size.
Analogy: Like a dabba (tiffin box) with adjustable compartments. On a big lunch plate (desktop), you have many compartments side by side. When you switch to a smaller dabba (mobile), the compartments automatically stack vertically or reduce in number per row. auto-fit is like "collapse empty compartments", while auto-fill is like "keep the empty space for potential items".

/* Responsive Grid Techniques */

/* 1. Auto-Fit - Stretches items to fill space */
.grid-autofit {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
    gap: 20px;
    /* Columns collapse when screen shrinks, items expand */
}

/* 2. Auto-Fill - Maintains track size */
.grid-autofill {
    display: grid;
    grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
    gap: 20px;
    /* Creates as many columns as fit, keeps empty tracks */
}

/* 3. Media Query Approach */
.grid-responsive {
    display: grid;
    gap: 20px;
}

/* Mobile First */
.grid-responsive {
    grid-template-columns: 1fr;  /* Single column */
}

/* Tablet */
@media (min-width: 768px) {
    .grid-responsive {
        grid-template-columns: repeat(2, 1fr);  /* Two columns */
    }
}

/* Desktop */
@media (min-width: 1024px) {
    .grid-responsive {
        grid-template-columns: repeat(4, 1fr);  /* Four columns */
    }
}

/* 4. Combined Approach - Best Practice */
.product-grid {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(min(250px, 100%), 1fr));
    gap: 20px;
    /* Prevents horizontal scroll on tiny screens */
}
