1. getElementById() - Finding Elements by ID

Concept: Retrieves a single element from the DOM using its unique ID attribute. It's the fastest way to access a specific element.
Think of your webpage as a classroom full of students. Using getElementById() is like calling out a student's roll number - you instantly get that ONE specific student because roll numbers are unique!

2. querySelector() - CSS Selector Power

Concept: Uses CSS selector syntax to find the FIRST matching element. More flexible than getElementById as it can search using classes, attributes, or complex selectors.
It's like asking a librarian: "Give me the first book on the third shelf from the Mystery section." You can be very specific with your search criteria using CSS rules!

// Different ways to use querySelector
function selectByClass() {
    // Select first element with class "mystery"
    const element = document.querySelector('.mystery');
    element.style.backgroundColor = '#ffeb3b';
}

function selectByAttribute() {
    // Select element with specific attribute
    const element = document.querySelector('[data-author="chetan"]');
    element.style.backgroundColor = '#4caf50';
    element.style.color = 'white';
}

function selectComplex() {
    // Select element that has BOTH classes
    const element = document.querySelector('.book.fiction');
    element.style.backgroundColor = '#f44336';
    element.style.color = 'white';
}

// HTML Structure
<p class="book mystery">Book 2: Sherlock Holmes</p>
<p class="book fiction" data-author="chetan">Book 3: Five Point Someone</p>

3. innerHTML vs textContent - Content Manipulation

Concept: Both change element content, but innerHTML parses HTML tags while textContent treats everything as plain text.
Imagine writing on a whiteboard. textContent is like writing plain text with a marker. innerHTML is like having magic markers that can draw shapes, colors, and formatted content when you write special codes!

// innerHTML - Parses and renders HTML tags
function useInnerHTML() {
    const element = document.getElementById('content-demo');
    element.innerHTML = '<strong>Chai</strong> and <em>Samosa</em>';
    // Result: Bold Chai and Italic Samosa
}

// textContent - Treats everything as plain text
function useTextContent() {
    const element = document.getElementById('content-demo');
    element.textContent = '<strong>Chai</strong> and <em>Samosa</em>';
    // Result: Literal text "Chai and Samosa"
}

‚ö†Ô∏è Security Warning: Never use innerHTML with user input directly! It can lead to XSS (Cross-Site Scripting) attacks. Always sanitize or prefer textContent for user-generated content.

4. Changing Styles - Dynamic Appearance Control

Concept: Modify CSS properties dynamically using JavaScript through the style property. Each CSS property becomes a camelCase JavaScript property.
Think of styling as dressing up for different occasions. Your webpage is like a person who can instantly change clothes (styles) based on events - formal for meetings (dark theme), casual for parties (bright theme), sporty for gym (dynamic animations)!

function makeItRed() {
    const element = document.getElementById('style-demo-box');

// Complete Solution
function generateGradeCard() {
    // 1. Get input values using querySelector
    const name = document.querySelector('#exercise-name').value;
    const marks = parseInt(document.querySelector('#exercise-marks').value);
    
    // 2. Get the grade card container using getElementById
    const gradeCard = document.getElementById('grade-card');
    
    // 3. Calculate grade
    let grade, remark, gradeColor;
    if (marks >= 90) {
        grade = 'A+'; remark = 'Outstanding! üéâ'; gradeColor = '#4caf50';
    } else if (marks >= 75) {
        grade = 'A'; remark = 'Excellent! üëè'; gradeColor = '#8bc34a';
    } else if (marks >= 60) {
        grade = 'B'; remark = 'Good job! üëç'; gradeColor = '#ff9800';
    } else if (marks >= 40) {
        grade = 'C'; remark = 'Pass'; gradeColor = '#ff5722';
    } else {
        grade = 'F'; remark = 'Needs Improvement'; gradeColor = '#f44336';
    }
    
    // 4. Use innerHTML to create formatted content
    gradeCard.innerHTML = `
        <h3>üìú Grade Card</h3>
        <p><strong>Name:</strong> ${name}</p>
        <p><strong>Marks:</strong> ${marks}/100</p>
        <p><strong>Grade:</strong> <span id="grade-display">${grade}</span></p>
        <p><strong>Remark:</strong> ${remark}</p>
    `;
    
    // 5. Change styles dynamically
    gradeCard.style.display = 'block';
    gradeCard.style.backgroundColor = gradeColor + '22'; // Add transparency
    gradeCard.style.border = `3px solid ${gradeColor}`;
    gradeCard.style.borderRadius = '10px';
    gradeCard.style.padding = '25px';
    
    // 6. Style the grade display specifically
    const gradeDisplay = document.getElementById('grade-display');
    if (gradeDisplay) {
        gradeDisplay.style.color = gradeColor;
        gradeDisplay.style.fontSize = '24px';
        gradeDisplay.style.fontWeight = 'bold';
    }
}

üéØ Key Takeaways:
  • getElementById is fastest for unique elements (IDs)
  • querySelector gives you CSS selector flexibility
  • innerHTML for HTML content, textContent for safety
  • style property uses camelCase (backgroundColor, not background-color)
  • Always validate and sanitize user input when using innerHTML
