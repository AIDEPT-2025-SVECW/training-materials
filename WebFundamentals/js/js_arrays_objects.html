1. Array Iteration Methods

Beyond basic loops: Modern JavaScript provides powerful array methods that make iteration cleaner and more functional.
Think of array methods like different Mumbai local train routes - each gets you to your destination, but some are express (map, filter) while others stop everywhere (forEach).

forEach vs map vs filter

// forEach - Does something with each item (no return)
const cities = ['Mumbai', 'Delhi', 'Bangalore', 'Chennai'];
cities.forEach(city => console.log(city));

// map - Transforms each item into something new
const upperCities = cities.map(city => city.toUpperCase());

// filter - Selects items based on condition
const longNames = cities.filter(city => city.length > 6);

2. Array Manipulation Methods

Mutating vs Non-Mutating: Some methods change the original array (mutating), others return a new array (non-mutating).

Mutating Methods (Change Original)

let snacks = ['Samosa', 'Vada Pav', 'Dhokla'];

// push/pop - Add/remove from end
snacks.push('Jalebi');    // ['Samosa', 'Vada Pav', 'Dhokla', 'Jalebi']
snacks.pop();              // Removes 'Jalebi'

// unshift/shift - Add/remove from start
snacks.unshift('Kachori'); // ['Kachori', 'Samosa', 'Vada Pav', 'Dhokla']
snacks.shift();            // Removes 'Kachori'

// splice - Add/remove from anywhere
snacks.splice(1, 1, 'Pani Puri'); // Replaces 'Vada Pav' with 'Pani Puri'

Non-Mutating Methods (Return New Array)

const states = ['Maharashtra', 'Gujarat', 'Karnataka'];

// slice - Extract portion
const firstTwo = states.slice(0, 2); // ['Maharashtra', 'Gujarat']

// concat - Join arrays
const moreStates = states.concat(['Tamil Nadu', 'Kerala']);

// spread operator - Modern way to copy/combine
const allStates = [...states, 'Punjab', 'Rajasthan'];

3. Advanced Array Methods

Searching & Testing: Find elements or check conditions across arrays efficiently.

find, findIndex, some, every

const products = [
    { name: 'Laptop', price: 45000, inStock: true },
    { name: 'Mobile', price: 15000, inStock: false },
    { name: 'Tablet', price: 25000, inStock: true }
];

// find - Returns first matching element
const laptop = products.find(p => p.name === 'Laptop');

// findIndex - Returns index of first match
const index = products.findIndex(p => p.price > 20000);

// some - Returns true if ANY element matches
const hasOutOfStock = products.some(p => !p.inStock); // true

// every - Returns true if ALL elements match
const allInStock = products.every(p => p.inStock); // false

4. Object Creation & Patterns

Multiple ways to create objects: Literal notation, constructors, Object.create(), and classes.
Creating objects is like filling out forms - object literal is a pre-printed form, constructor is a template you can reuse, and classes are modern digital forms with validation built-in.

Object Literal (Simple & Direct)

// Best for single objects
const student = {
    name: 'Rahul Sharma',
    rollNo: 101,
    course: 'B.Tech CSE',
    getInfo() {
        return `${this.name} - ${this.rollNo}`;
    }
};

Constructor Function (Reusable Template)

// Create multiple similar objects
function Student(name, rollNo, course) {
    this.name = name;
    this.rollNo = rollNo;
    this.course = course;
    this.getInfo = function() {
        return `${this.name} - ${this.rollNo}`;
    };
}

const student1 = new Student('Priya', 102, 'B.Tech ECE');
const student2 = new Student('Amit', 103, 'B.Tech ME');

ES6 Classes (Modern Approach)

class Student {
    constructor(name, rollNo, course) {
        this.name = name;
        this.rollNo = rollNo;
        this.course = course;
    }
    
    getInfo() {
        return `${this.name} - ${this.rollNo}`;
    }
    
    static totalStudents = 0; // Static property
}

const student1 = new Student('Neha', 104, 'B.Tech IT');

5. Object Methods & Manipulation

Working with objects: Access keys, values, entries, and manipulate object properties dynamically.

Object.keys(), Object.values(), Object.entries()

const restaurant = {
    name: 'Taj',
    cuisine: 'North Indian',
    rating: 4.5,
    city: 'Mumbai'
};

// Get all keys (property names)
Object.keys(restaurant); 
// ['name', 'cuisine', 'rating', 'city']

// Get all values
Object.values(restaurant); 
// ['Taj', 'North Indian', 4.5, 'Mumbai']

// Get key-value pairs
Object.entries(restaurant); 
// [['name', 'Taj'], ['cuisine', 'North Indian'], ...]

Spreading & Merging Objects

const basicInfo = { name: 'Rajesh', age: 22 };
const contactInfo = { email: 'rajesh@example.com', phone: '9876543210' };

// Spread operator - Merge objects
const completeInfo = { ...basicInfo, ...contactInfo };

// Object.assign() - Alternative way
const merged = Object.assign({}, basicInfo, contactInfo);

// Shopping Cart using Array of Objects
const cart = [];

function addToCart(name, price) {
    const item = {
        id: Date.now(),
        name: name,
        price: price,
        quantity: 1
    };
    
    // Check if item already exists
    const existing = cart.find(item => item.name === name);
    if (existing) {
        existing.quantity++;
    } else {
        cart.push(item);
    }
}

function calculateTotal() {
    // Using reduce to sum all prices
    return cart.reduce((total, item) => {
        return total + (item.price * item.quantity);
    }, 0);
}

7. Method Chaining

Combining methods: Chain multiple array methods together for powerful data transformations.
Method chaining is like an assembly line in a factory - each step transforms the product before passing it to the next station. 
const scores = [45, 78, 92, 34, 88, 61, 95, 73];

// Chain multiple methods
const result = scores
    .filter(score => score >= 60)      // Only passing scores
    .map(score => score + 5)           // Grace marks
    .sort((a, b) => b - a)             // Sort descending
    .slice(0, 3);                      // Top 3

console.log(result); // [100, 93, 83]
