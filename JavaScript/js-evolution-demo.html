The Birth: Mocha/LiveScript

üìÖ 1995 - September

JavaScript was created by Brendan Eich in just 10 days at Netscape. Originally called "Mocha", then "LiveScript", it was finally renamed "JavaScript" for marketing reasons to capitalize on Java's popularity.

Key Features: Basic scripting, simple DOM manipulation, form validation

// Original JavaScript (1995)
// Simple variable declaration
var greeting = "Hello, World!";

// Basic function
function showAlert() {
    alert(greeting);
}

// Simple DOM manipulation
document.write("Welcome to JavaScript!");

ES3: The Standardization

üìÖ 1999 - December

ECMAScript 3 brought regular expressions, better string handling, try/catch exception handling, and more. This version dominated for a decade.

// ES3 Features (1999)
// Regular Expressions
var emailPattern = /^[\w-\.]+@([\w-]+\.)+[\w-]{2,4}$/;
var email = "user@example.com";
var isValid = emailPattern.test(email);

// Try-Catch Exception Handling
try {
    var result = riskyOperation();
} catch(error) {
    console.log("Error: " + error.message);
}

// Array methods
var numbers = [1, 2, 3, 4, 5];
var doubled = [];
for(var i = 0; i < numbers.length; i++) {
    doubled.push(numbers[i] * 2);
}

ES5: The Modern Foundation

üìÖ 2009 - December

ES5 introduced strict mode, JSON support, array methods (forEach, map, filter), and Object methods. This version made JavaScript more robust and developer-friendly.

// ES5 Features (2009)
"use strict"; // Strict mode

// Array methods
var numbers = [1, 2, 3, 4, 5];

// map - transform array
var doubled = numbers.map(function(num) {
    return num * 2;
});

// filter - select elements
var evens = numbers.filter(function(num) {
    return num % 2 === 0;
});

// forEach - iterate
numbers.forEach(function(num) {
    console.log(num);
});

// JSON support
var person = {name: "John", age: 30};
var jsonString = JSON.stringify(person);
var parsed = JSON.parse(jsonString);

// ES6 Features (2015)
// Let and Const
const PI = 3.14159;
let counter = 0;

// Arrow Functions
const multiply = (a, b) => a * b;
const square = x => x * x;

// Template Literals
const name = "JavaScript";
const version = "ES6";
const message = `Welcome to ${name} ${version}!`;

// Destructuring
const user = {firstName: "John", lastName: "Doe", age: 30};
const {firstName, lastName} = user;

const colors = ["red", "green", "blue"];
const [first, second] = colors;

// Classes
class Person {
    constructor(name, age) {
        this.name = name;
        this.age = age;
    }
    
    greet() {
        return `Hello, I'm ${this.name}`;
    }
}

// Promises
const fetchData = () => {
    return new Promise((resolve, reject) => {
        setTimeout(() => resolve("Data loaded!"), 1000);
    });
};

Modern JavaScript (ES2016+)

üìÖ 2016 - Present

JavaScript now receives yearly updates. Major additions include async/await, optional chaining, nullish coalescing, and more.

// Modern JavaScript Features
// Async/Await (ES2017)
async function loadData() {
    try {
        const response = await fetch('api/data');
        const data = await response.json();
        return data;
    } catch(error) {
        console.error(error);
    }
}

// Optional Chaining (ES2020)
const userName = user?.profile?.name ?? 'Guest';

// Nullish Coalescing (ES2020)
const port = config.port ?? 3000;

// Array methods
const numbers = [1, 2, 3, 4, 5];

// includes (ES2016)
const hasThree = numbers.includes(3);

// flat (ES2019)
const nested = [1, [2, [3, 4]]];
const flat = nested.flat(2);

// Object.entries (ES2017)
const obj = {a: 1, b: 2};
Object.entries(obj).forEach(([key, value]) => {
    console.log(`${key}: ${value}`);
});
