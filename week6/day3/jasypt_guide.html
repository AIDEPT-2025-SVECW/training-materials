What is JASYPT?

JASYPT (Java Simplified Encryption) is a Java library that allows developers to add basic encryption capabilities to projects with minimum effort. It's particularly useful for encrypting sensitive configuration properties like database passwords.

‚ö†Ô∏è Security Problem

Hard-coded passwords in application.properties are a major security risk. Anyone with access to your code can see sensitive credentials!

Benefits of JASYPT:

  • üîí Encrypts sensitive properties
  • üîë Uses master password for decryption
  • üåç Environment variable support

Command Line Encryption

You can also encrypt using JASYPT CLI:

# Download JASYPT CLI wget https://github.com/jasypt/jasypt/releases/download/jasypt-1.9.3/jasypt-1.9.3-dist.zip # Encrypt a password java -cp jasypt-1.9.3.jar org.jasypt.intf.cli.JasyptPBEStringEncryptionCLI \ input="mySecretPassword123" password="myMasterKey123" algorithm="PBEWithMD5AndDES"

Step 3: Spring Boot Integration

3

Update application.properties

Replace plain passwords with encrypted values using ENC(...) syntax:

# application.properties - SECURE! spring.datasource.url=jdbc:mysql://localhost:3306/mydb spring.datasource.username=admin spring.datasource.password=ENC(your_encrypted_password_here) spring.mail.password=ENC(your_encrypted_email_password_here) # JASYPT Configuration jasypt.encryptor.password=${JASYPT_ENCRYPTOR_PASSWORD} jasypt.encryptor.algorithm=PBEWithMD5AndDES
4

Set Environment Variable

# Linux/Mac export JASYPT_ENCRYPTOR_PASSWORD=myMasterKey123 # Windows set JASYPT_ENCRYPTOR_PASSWORD=myMasterKey123 # Or pass as JVM argument java -Djasypt.encryptor.password=myMasterKey123 -jar myapp.jar
5

Configuration Class (Optional)

@Configuration @EnableEncryptableProperties public class JasyptConfig { @Bean("jasyptStringEncryptor") public StringEncryptor stringEncryptor() { PooledPBEStringEncryptor encryptor = new PooledPBEStringEncryptor(); SimpleStringPBEConfig config = new SimpleStringPBEConfig(); config.setPassword(System.getenv("JASYPT_ENCRYPTOR_PASSWORD")); config.setAlgorithm("PBEWithMD5AndDES"); config.setKeyObtentionIterations("1000"); config.setPoolSize("1"); config.setProviderName("SunJCE"); config.setSaltGeneratorClassName("org.jasypt.salt.RandomSaltGenerator"); config.setStringOutputType("base64"); encryptor.setConfig(config); return encryptor; } }

üö® Security Best Practices

  • Never commit master passwords to version control
  • Use different master passwords for different environments
  • Store master passwords in secure key management systems
  • Use strong encryption algorithms in production
  • Regularly rotate your master passwords
  • Consider using cloud-native secret management services
