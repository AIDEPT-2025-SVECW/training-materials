Java Database Connectivity (JDBC)

JDBC is a Java API that enables Java applications to interact with databases. It provides a standard interface for connecting to and executing queries against various database systems.

Key Features:

  • Database-independent connectivity
  • Standard SQL support
  • Transaction management
  • Result set processing
  • Exception handling

üöó JDBC Drivers

What is a JDBC Driver?

A JDBC driver is a software component that enables Java applications to interact with databases. It acts as a translator between Java code and database-specific protocols.

üåâ Type 1: JDBC-ODBC Bridge

Translates JDBC calls to ODBC calls

Status: Deprecated in Java 8+

üîß Type 2: Native-API Driver

Uses database-specific native libraries

Performance: Better than Type 1

üåê Type 3: Network Protocol

Uses middleware server for communication

Advantage: Database independent

‚ö° Type 4: Pure Java Driver

Direct communication with database

Most Popular: Platform independent
